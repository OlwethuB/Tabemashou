API Descriptions

HTTP Method Path Description
Home
Get /(home page) This directs us to the the 'front' page, the very first thing we see when we open or run the Api.
Users
Get /users Display a list of users
Get /user/:id Display a single user
Post /register To register a new user
Post /login To login the user
Post /logout To logout the user
Patch/Put /users/:id To modify a user’s record OR Update user’s record
Delete /user/:id To delete a single user
Products
Get /products Display a list of products
Get /products/:id Display a single product
Post /products To add a new product
Put/Patch /products/:id Modify or replace a product’s record
Delete /products/:id Delete a product
Bookings
Get /bookings Display a list of reservations
Get /booking/:id Display a single product
Post /book To add a new product
Put/Patch /booking/:id Modify or replace a product’s record
Delete /booking/:id Delete a product
